Задание 4: CLI. 2 балла
Зачем вы обычно исполняете скрипты php в режиме cli?

- Я использую php cli для выполнения скриптов которые я могу запустить из командной строки.
Это могут быть скрипты для обработки, создания, удаления файлов и т.п.
Часто эти скрипты выполняют длительную работу и могут быть запущены другими программами на сервере, например через cron. 
- Я так же использую этот режим для запуска php web сервера. php -S 0.0.0.0:8085
- В Laravel есть artisan он работает в консоле им я часто пользуюсь.
- Иногда я использую php в интерактивном режиме. php -a, или для выполнения выражения через php -r "phpinfo();"
